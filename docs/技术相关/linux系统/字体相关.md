# 字体相关
# 更新整个系统的字体缓存
``` shell
sudo fc-cache -fv
```

# 将自己的字体添加到整个系统
进入这个路径，将自己的字体加入
``` shell
/usr/local/share/fonts
```
举例：
``` shell
zane@debian:/usr/local/share/fonts$ tree -L 2
```
